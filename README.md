# PROG5121 POE - Part 3 

## Description
A Java console application that allows users to register, login, send and manage messages.

## Features
- User registration and login
- Send messages with validation
- Message hash generation
- Store messages to JSON file
- Display stored messages
- Search messages by ID or recipient
- Delete messages by hash
- Display full report of all messages
- Unit tests for all methods

## How to Run
1. Open project in IntelliJ
2. Run Main.java
3. Register a user then login
4. Use the menu to send and manage messages

## Technologies Used
- Java 17
- Maven
- json-simple 1.1.1
- JUnit 5
